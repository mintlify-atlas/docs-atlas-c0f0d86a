# Welcome to quantumbagel.github.io

This is the documentation site for Julian Reder's personal developer portfolio, hosted at [quantumbagel.github.io](https://quantumbagel.github.io). The portfolio is version 2, rebuilt from the ground up with TailwindCSS and driven by a Python generator script.

<Info>
  The entire portfolio's content — projects, skills, experience — lives in a single Python file (`generator.py`). Updating the site is as simple as editing that file and running `python generator.py`.
</Info>

## Key features

- **Python-powered generator** — all data lives in `generator.py`; the HTML is built on demand
- **Dark mode** — automatic detection of user OS preference, fully styled for both modes
- **Interactive project cards** — modal detail view with feature lists, skill tags, and GitHub links
- **Category filtering** — filter projects by type: Discord Bots, Internship, or Other
- **Expandable skill bars** — click any skill to reveal details and related projects
- **Responsive layout** — fixed sidebar on desktop, stacked on mobile
- **GitHub Pages deployment** — minified HTML output via `htmlmin`; commit and push to deploy
