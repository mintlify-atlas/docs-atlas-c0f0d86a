# About Julian Reder

Hello! I'm Julian Reder, a programmer and student at North Carolina State University studying Computer Engineering. This portfolio is version 2 — rebuilt entirely from scratch using Tailwind CSS for styling and layout, and driven by a Python generator script.

This site serves as a list of the projects I've made. Feel free to browse my projects or learn more about my experience. If you want to get in touch, use the contact form at the bottom of the [live portfolio](https://quantumbagel.github.io) or reach out via one of the social links below.
